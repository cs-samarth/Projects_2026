HTTP

Http is an application layer protocol that defines how applications exchange information over a network, relying on lower-layer protocols to transport the data. A typical flow over HTTP involves a client machine making a request to a server, which then sends a response message.

What is an http request

Http request is a way Internet communication platforms like web browsers ask for the information they need to load a website. Each HTTP request made across the Internet carries with it a series of encoded data that carries different types of information. A typical HTTP request contains.

  1. HTTP version type
  2. a URL
  3. an HTTP method
  4. HTTP request header
  5. Optional HTTP body

DNS

Domain name system is hierarchical and decentralized naming system for internet connected resources.

The process of DNS resolution involves converting a hostname into a computer-friendly IP address.

DNS Servers involved in loading a webpage

  • DNS recursor
  • Root nameserver
  • TLD nameserver
  • Authoritative nameserver
